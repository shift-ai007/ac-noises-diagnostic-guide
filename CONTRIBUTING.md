# Contributing

We welcome contributions to improve this AC noise diagnostic guide. Whether you're an HVAC technician with field experience or a homeowner who figured something out the hard way, your input helps everyone.

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your change (`git checkout -b add-noise-type`)
3. **Make your changes** — keep language accessible for homeowners
4. **Submit a pull request** with a clear description of what you added or fixed

## What We're Looking For

- Additional noise types and their causes
- Corrections to diagnostic information
- Regional variations (different climates, system types)
- Better descriptions of sounds (audio references are great)
- Cost estimates for common repairs

## Guidelines

- Write for homeowners, not technicians — avoid jargon or explain it when used
- Include severity ratings (Low / Medium / High) for each noise type
- Mention when professional help is needed vs. safe DIY fixes
- All information should be accurate and based on real HVAC experience
